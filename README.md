# 🎨 Figurine Auto-Installer for Debian

Este repositorio contiene un script de automatización en Bash para instalar y configurar **Figurine** de forma rápida en sistemas basados en Debian.

## ✨ Características
* Descarga automática del binario desde GitHub.
* Instalación en `/usr/local/bin/`.
* Configuración automática del archivo `.bashrc`.
* Muestra el `hostname` del sistema con un diseño estilizado al iniciar sesión.

## 🚀 Instalación Rápida

Para instalar todo con un solo comando (vía cURL), ejecuta:

##```bash
## curl -sSL [https://raw.githubusercontent.com/TU_USUARIO/TU_REPO/main/instalar_figurine.sh](https://raw.githubusercontent.com/TU_USUARIO/TU_REPO/main/instalar_figurine.sh) | bash
